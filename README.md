# Banking System (C)

## Overview
A console-based banking application that simulates basic financial operations such as account management and transactions.

## Features
- User registration and login
- Deposit and withdrawal functionality
- Fund transfer between users
- Balance inquiry
- Transaction history tracking

## Technologies Used
- C Programming
- File Handling
- String Manipulation

## Project Structure
- src/ → main source code
- data/ → stores account and transaction data

## How to Run
gcc src/main.c
./a.out
